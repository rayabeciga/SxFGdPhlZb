# 项目简介

## 核心功能
- 基于YOLOv8的药品目标检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同模型进行检测
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 药品识别与管理
- 医药仓库盘点
- 药店自动化管理
- 医疗机构的药品分类与统计
- 药品生产线的质量控制

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发
- 环境配置：Python3.9、opencv-python、PyQt5、ultralytics、torch1.9
- 包含6028张药品数据集，涵盖244种药品名称
- 提供完整源码、数据集、训练好的模型及项目运行教程
- 配置成功环境后可直接使用
